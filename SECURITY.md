# Security Policy

## Reporting a Vulnerability
If you believe you have discovered a security vulnerability in OnPremPDF,
please report it responsibly by contacting:

info@onprempdf.com

Please do not disclose security issues publicly before we have had a chance
to investigate and address them.

## Response Process
We aim to acknowledge security reports within 2 business days and provide
an initial assessment within 5 business days.

## Scope
This security policy applies to:
- The OnPremPDF application
- Official installation scripts and documentation

It does not apply to customer infrastructure, network configuration,
or third-party software not distributed by OnPremPDF.
