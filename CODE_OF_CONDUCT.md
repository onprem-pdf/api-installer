# Code of Conduct

This project aims to provide a professional and respectful environment for
technical discussion and usage.

## Expected Behavior
- Be respectful and constructive
- Keep discussions technical and on-topic
- Assume good intent

## Unacceptable Behavior
- Harassment, insults, or personal attacks
- Discriminatory or abusive language
- Intentionally disruptive behavior

Project maintainers may remove content or restrict participation that violates
these guidelines.

This Code of Conduct applies to all interactions related to this repository.
